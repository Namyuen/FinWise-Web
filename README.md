# FinWise — Financial Literacy for Students

FinWise is a static educational website designed to improve financial literacy by providing learning resources, interactive quizzes, and practical financial tools.

---

## Motivation

This project was created to make financial education more accessible through a simple, user-friendly web application.

## Technologies Used

HTML5
CSS3
JavaScript


---

## ✨ Features

| Feature | Details |
|---|---|
| **Learn** | 5 modules × 17 lessons — each with explanation, real-world example, key takeaways, and a mini-quiz |
| **Quiz** | 20-question comprehensive quiz with per-question explanations and a scored results screen |
| **Budget Calculator** | Monthly income vs. expenses with savings rate progress bar |
| **Compound Interest Calculator** | Animated Chart.js growth chart with sliders for rate and time period |
| **Savings Goal Calculator** | Calculates months to target with a live progress indicator |
| **Resources** | Government portals, learning platforms, recommended books, and free tools |
